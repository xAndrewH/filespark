# FileSpark

FileSpark is a free, browser-based file converter and tools platform. No uploads, no accounts — everything runs in your browser.

**Live:** [filespark.app](https://filespark.app)

## Tools

FileSpark includes 85+ tools across the following categories:

- **Image conversion** — convert between PNG, JPG, WebP, AVIF, SVG, and more
- **Video & audio** — transcode, compress, trim, extract audio
- **Documents** — convert PDFs, Word docs, spreadsheets, and ebooks
- **Fonts & archives** — convert font formats, zip/unzip files
- **Text & code** — formatters, encoders, diff tools, minifiers
- **Design & color** — color pickers, palette generators, gradient tools
- **SEO & web** — meta tag generators, sitemap tools, Open Graph previews
- **Calculators** — unit converters, date calculators, and more

## Development

npm run dev      # start the development server at http://localhost:3000
